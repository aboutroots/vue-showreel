## Development

Requirements: `node 16+`

```bash
# Install dependencies:
yarn install

# To compile and hot-reload for development:
yarn serve

# To compiles and minify for production:
yarn build

# if you want to serve the production build locally, run:
yarn serve-static

# To lint and fix files:
yarn lint

# To run unit tests:
yarn test
```
