## Development

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

if you want to serve the production build locally, run

```
yarn serve-static
```

### Lints and fixes files

```
yarn lint
```

### Unit tests

```
yarn test
```
